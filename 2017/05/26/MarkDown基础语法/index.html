概述

目前，市面上有大量MarkDown的衍生版本，csdn，有道云等都有丰富的扩展。这些衍生版本，都对应有自己的扩展语法，并且各有不同。所以，这里只介绍基础的MarkDown语法。同时也建议大家使用基础语法，因为任何衍生版本，都基于此。

如果你发现根据讲述的语法，并没有达到应有的效果，请检查输入法是否为中文。

markdown语法，仅针对英文字符，其他语言无效。

工具：MarkDownPad2,链接：http://markdownpad.com/

段落和行

一个段落由一个或多个连续文本行组成。也就是说，在编辑时，即使看起来是两行，但实际上markdown认为这也是同一个段落。

相邻的一行是为同一段落		 第一行相邻的一行为同一段落

两行之间存在两个空行即为不同段落		 第一行

表格

首先需要强调，Markdown原生语法是不支持表格的。但几乎所有衍生语法都支持，切表格语法一致。

MarkDown:
| 默认 | 靠右 | 居中 | 靠左 |
| —- | —: | :–: | :— |
| 内容 | 内容 | 内容 | 内容 |
| 内容 | 内容 | 条目 | 内容 |

###预览
